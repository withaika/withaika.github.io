NXT

The NXT object is used to manage the connection between your Android phone and the NXT brick. Usually you create one these objects at the start of you program like this:

 nxt = app.CreateNxt();

See the CreateNxt function for more info.

The following functions are provided by the NXT object:

  • Beep
  • Brake
  • CheckConnection
  • Disconnect
  • Drive
  • GetCurrentProgram
  • IsBluetoothEnabled
  • IsMotorIdle
  • PlaySoundFile
  • ReadColorSensor
  • ReadDistanceSensor
  • ReadMail
  • ReadLightSensor
  • ReadTouchSensor
  • ReadSoundSensor
  • SendMail
  • SetInvert
  • SetLampColor
  • SetOnConnected
  • ShowDevices
  • StartProgram
  • Stop
  • StopProgram
