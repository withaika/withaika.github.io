CreateNxt

The CreateNxt function creates an object which manages the connection between your Android phone and the NXT brick.

 nxt = app.CreateNxt();

Once it is created you can use the ShowDevices function which will allow the user to select from a list of NXT bricks which have been paired with the phone.

Example - Make the NXT beep

function OnStart()
{
  nxt = app.CreateNxt();
   app.SetMenu( "Connect,Beep" );
}

function OnMenu( name )
{
  if( name=="Connect" )
    nxt.ShowDevices();
  else if( name=="Beep" )
    nxt.Beep( 100, 500 );
}

You can optionally use the SetOnConnected function to set a callback function which will be called when the brick is successfully connected to your phone via Bluetooth.

You may also want to use the CheckConnection function at various times to check you still have a connection to the NXT brick.
