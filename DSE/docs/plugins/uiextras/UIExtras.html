UIExtras

The UIExtras plugin is a collection of User Interface components (controls and dialogs) to enhance your DroidScript Apps.

In order to use UIExtras, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "UIExtras" );

Then you can create an instance of the plugin object when you need it like this:

 uix = app.CreateUIExtras();

The following UI components are provided by the UIExtras plugin:

Controls

  • Picker
  • NumberPicker
  • DatePicker
  • TimePicker
  • FloatingActionButton
  • FloatingActionsMenu

Dialogs

  • DatePickerDialog
  • TimePickerDialog
  • ColorPickerDialog
