ReadMail

The ReadMail function reads a message from the NXT brick's mail box. This message can be written using a normal NXT-G program running on the brick. This allows you to read values from NXT-G programs with your phone or tablet.

The mailbox parameter is the target mailbox number and can be a value between 1 and 10.

The type parameter should be one of the following values:- "Text", "Number" or "Logic" depending on what type of value you wish to read from the brick's mailbox.

The remove parameter should be true if you wish to remove the message from the NXT's mailbox after reading it or false if you wish to leave the message in the mailbox.

 nxt.ReadMail( mailbox, type, remove );

Example - Read number from mailbox 1

function OnStart()
{
  nxt = app.CreateNxt();
  app.SetMenu( "Connect,ReadMail" );
}

function OnMenu( name )
{
  if( name=="Connect" )
    nxt.ShowDevices();
  else if( name=="ReadMail" )
    nxt.ReadMail( 1, "Number", true );
 }
